This was a project created by me, Chris Yeung, Dylan Nguyen, and Brady Zhang during Hoya Hacks January 2018.

Our idea was to scrape subreddit data for the top 6 English Premier League soccer teams over time, use natural language processing to determine the average sentiment per week, and then go from there

It was the first Hackathon any of us had went to, so we were pretty excited. Brady was also the only person who knew Django, which we had decided on as our web framework. Then Brady got sick and couldn't really work on things so... yeah.

At first we wanted the data to be scraped and analyzed on the fly because, that would be pretty cool and we could do virtually any subreddit then. It turns out this kind of thing can take a long time, even if you take shortcuts like only analyzing the top 20 comments, sampling less posts per week etc. We also ran into usage limits with PRAW we weren't aware of which also stalled our progress since we thought something was wrong somewhere with our code, so we kept running things again, continually making sure we were at the PRAW api limits. Oops.

We pretty much ended up using Django to show one webpage, which displayed pre-scraped data from reddit which we stored into a csv. We stored it in a csv in a vain effort to make it play nicer with NVD3. Naturally it didn't.

All in all I felt pretty meh about our project back then but looking back now I'm very happy with how things went. It was really fun, despite almost nothing going as planned.
